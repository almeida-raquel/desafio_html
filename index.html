Ferramentas utilizadas

Neste curso, a ferramenta utilizada para desenvolvimento web é o Visual Code Studio. O VS Code, como também é conhecido, é open source e de fácil instalação em qualquer sistema operacional.

Além disso, para iniciar a edição de textos é importante fazer também o download da extensão Live Server, para aumentar as funcionalidades do Visual Studio Code e adequá-lo para o desenvolvimento da página web.

Inspetor de elementos

A ferramenta inspetor de elementos é fornecida pelo próprio navegador que está sendo utilizado, podendo ser acessada através do comando “F12” no teclado ou também através da opção mais ferramentas na barra superior do navegador. Este console oferece vários recursos que mostram como a página foi feita. É possível também realizar algumas alterações na forma como o site está sendo exibido exclusivamente no seu browser.

Estrutura básica do HTML

Com a ferramenta VS Code configurada, cria-se um arquivo chamado “index.html” para iniciar o processo de desenvolvimento. No HTML, utilizam-se tags de marcação para estruturar os textos que serão escritos na montagem da página.

A base principal é formada pelas tags html, head e body. A tag head permite que informações sejam carregadas antes de exibir para o usuário, já a tag body exibirá para o usuário o que estiver delimitado por ela.

Falando sobre tags

Existem tags responsáveis por alterar a aparência dos textos que estão sendo escritos, são elas:

i - texto em itálico;

strong - texto em negrito;

input type= “”> - cria campos para que informações sejam digitadas, sendo possível escolher o tipo de informação que será digitada neste campo.

Atributos das tags

Tratam-se das propriedades que uma tag pode possuir.

É possível alterar especificamente o conteúdo de uma parte do código utilizando “id”. Existe também uma maneira de formatar um código CSS, chama-se “style”. Através de sua utilização, é possível alterar a aparência de um código, mudando cor e tipo de fonte. Utilizando o comando “class” permite padronizar tudo o que estiver denominado com o mesmo título. O atributo img src=””/ permite que imagens sejam adicionadas à página, sendo possível configurar a dimensão da imagem adicionada através da utilização do width.

Textos

Atributos de texto

Cabeçalhos

HTML possui cabeçalhos que são representados pela letra “h”. Existem as tags h1 até h6, cada uma representa um nível de cabeçalho, sendo o mais importante o representado pela tag “h1”.

Parágrafos

O corpo do texto que será escrito é representado pela tag p, que se refere a parágrafos.

Citações

É possível inserir citações no texto através da tag blockquote; sendo assim, o que estiver entre estas tags será uma citação. Por exemplo:

"O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos."

Listas ordenadas e não ordenadas

Temos duas opções de listas, as ordenadas e as não ordenadas. Para que os itens inseridos nas listas sejam identificados, é necessário utilizar a tag li.

Listas ordenadas

São representadas pelo elemento ol, terão seus itens ordenados por uma contagem que os precede, sendo em numerais ou algarismos romanos. Por exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não ordenadas

São representadas pelo elemento ul, terão seus itens precedidos por marcadores, não apresentando contagem ou ordenamento. Por exemplo:

  • Item 1
  • Item 2
  • Item 3
